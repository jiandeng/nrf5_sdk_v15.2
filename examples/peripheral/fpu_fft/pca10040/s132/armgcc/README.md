# CMSIS-DSP FFT算法分析
## 基本信息
* 量程: ±4g
* 采样率: 25Hz
* 分辨率: 1000/1024mg

## 时间/频域分辨率
* 频率分辨率 = 采样率 / N = 25 / N (Hz)
* 时间分辨率 = N / 采样率 (s)

| N   | 分析周期(s) | 频率分辨率(Hz) |

| 64  | 2.5         | 0.4            |
| 128 | 5           | 0.2            |

| 512 | 20          | 0.05           |
| 1024| 40          | 0.25           |
